プライバシーポリシー (Privacy Policy)

最終更新日: 2025年9月7日

本プライバシーポリシーは、「Twitch User Log Viewer」（以下、「本拡張機能」といいます）が収集する情報、その利用方法、およびユーザーの権利について説明するものです。

1. 収集する情報

本拡張機能は、その機能を提供するために、以下の情報を収集・利用します。

  • Twitchアカウント情報:
    • 情報: TwitchユーザーID、表示名、プロフィール画像URL
    • 目的: ログイン状態の表示、およびどのユーザーのチャットログを表示しているかを明確にするために利用します。
  • 認証情報:
    • 情報: Twitch API連携のためのアクセストークンおよびリフレッシュトークン
    • 目的: ユーザーに代わってTwitch APIに安全にアクセスし、チャットログなどの情報を取得するために必要です。これらの情報はユーザーのローカルコンピュータ上の安全な領域（chrome.storage）にのみ保存され、パスワード自体を収集することはありません。
  • 個人的コミュニケーション:
    • 情報: ユーザーがTwitch上で投稿した公開チャットメッセージの履歴
    • 目的: 本拡張機能の中核機能である、特定のユーザーのチャットログを検索・表示するために利用します。
  • 支払いに関する情報:
    • 情報: プレミアム機能の利用状況（有効か無効か）
    • 目的: プレミアム機能が利用可能かどうかを判断するために利用します。クレジットカード番号や具体的な取引履歴といった決済情報は収集・保存しません。
  • ユーザーアクティビティ:
    • 情報: 本拡張機能内での操作履歴（お気に入り登録、閲覧履歴など）
    • 目的: お気に入り機能や「最後に開いたユーザー」機能を提供し、利便性を向上させるために利用します。これらの情報はユーザーのローカルコンピュータ上にのみ保存されます。
  • サーバーログ:
    • 情報: IPアドレス
    • 目的: 本拡張機能がTwitch APIと通信するために利用する外部サーバー（プロキシサーバー）では、サービスの安定稼働およびセキュリティ維持のため、アクセス元のIPアドレスが一時的に記録される場合があります。

2. 収集しない情報

本拡張機能は、以下の情報を意図的に収集することはありません。

  • 氏名、住所、年齢などの個人識別情報
  • 健康、医療に関する情報
  • パスワード、セキュリティ保護用の質問、PIN番号
  • Twitch外のウェブサイト閲覧履歴
  • メールやダイレクトメッセージの内容

3. 情報の利用目的

収集した情報は、以下の目的のために利用されます。

  • Twitchチャットログの検索・表示機能の提供
  • ログイン認証およびセッションの維持
  • お気に入り、閲覧履歴などのパーソナライズ機能の提供
  • サービスのセキュリティ維持および不正利用の防止

4. 情報の保管と第三者提供

  • Twitchアカウント情報、認証情報、ユーザーアクティビティに関する情報は、ユーザーのコンピュータ上のChromeストレージに安全に保管されます。
  • 法的な要請がある場合を除き、ユーザーの同意なくして収集した情報を第三者に販売、貸与、または開示することはありません。
  • チャットログ取得のため、本拡張機能は外部のプロキシサーバー（https://4u2w8ragph.execute-api.ap-northeast-1.amazonaws.com）を経由してTwitch APIと通信します。この通信は暗号化されています。

5. ユーザーの権利

ユーザーはいつでも本拡張機能からログアウトし、chrome.storageに保存された認証情報を削除することができます。また、本拡張機能をアンインストールすることで、ローカルに保存されたすべての情報が削除されます。

6. プライバシーポリシーの変更

本プライバシーポリシーは、法令の変更やサービスのアップデートに伴い、事前の通知なく改定されることがあります。重要な変更があった場合は、適切な方法でお知らせします。
